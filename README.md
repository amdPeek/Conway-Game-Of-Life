# Conway-Game-Of-Life

# requirements
  `pip install kivy`

# how to use
  First make scripts executable : 
  `chmod +x conway_gui.py` and `chmod +x conway_cmd.py`

  To run cmd style : 
  `./conway_cmd.py <size_grid> <iterative|thread_one|thread_two>`

  To run gui style : 
  `./conway_gui.py`
